# Loan Default Risk Analysis Dashboard

## 📊 Project Overview
This project focuses on analyzing loan data to identify key factors influencing loan default risk.

## 🎯 Objective
To identify high-risk customers using financial and behavioral factors.

## 📁 Dataset Description
Includes:
- Credit Score
- Income
- Loan Amount
- DTI
- Late Payments
- Payment Burden
- Risk Category
- Default Status

## 🛠 Tools Used
- Power BI
- DAX
- Excel / CSV

## 📊 Key Insights
- High DTI increases default risk
- Late payments increase default probability
- Low credit score leads to higher defaults
- High EMI burden increases risk

## 💡 Conclusion
Helps banks identify high-risk customers and improve decisions.
